Comparisions Between HTMl and WordPress

After using both WordPress and HTML, I've come to the conclusion that I much prefer coding my own website. I do like Squarespace, but WordPress on the other hand is much to clunky for my liking. Furthermore, when I do my own coding I have much more control over the look of my website. I found customization especially hard on WordPress, but I was able to quickly customize my website if I could access the HTML and CSS. I think that websites like WordPress are great for small projects, but I would suggest using Squarespace (or even better), coding it yourself for larger projects. I will say however, adding images is much easier on WordPress than HTML.
